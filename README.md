2) Second One is [Numpy Examples.ipynb](https://github.com/sushilm2011/MLCC-Kolkata/blob/master/Numpy%20Examples.ipynb)  
consists of [numpy](http://www.numpy.org/) examples.
3) Third One is [Numpy Exercises .ipynb](https://github.com/sushilm2011/MLCC-Kolkata/blob/master/Numpy%20Exercises%20.ipynb)  
complete this small exercise to test your knowledge and learnings (Feel Free to Google).
